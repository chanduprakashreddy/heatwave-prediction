📘 How It Works  ▸

01

Ensemble Forecasting

Gradient-boosted XGBoost + SARIMA time series model averaging for robust, recursive multi-step temperature prediction.

02

Climatology Baseline

30-year day-of-year temperature average with 15-day rolling window smoothing defines the seasonal normal for each calendar day.

03

Anomaly Detection

Combined Z-score (>2σ from climatological normal) and multi-feature Isolation Forest for robust statistical anomaly identification.

04

IMD Heatwave Criteria

India Meteorological Department standards: Plains ≥40°C + 4.5°C departure; Coastal ≥37°C + 4.5°C departure; ≥3 consecutive days. Severe: >6.4°C departure.
